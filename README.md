## Simple ETH/DAI stoploss script to Uniswap exchange

Express server for web3-js interaction with UniswapV2 exchange contract.

## Install

- Configure eth account to connect to Metamask wallet
- Web3
- Infura node key

## Dependencies

- Node v14
- Web3
- Infura node key
